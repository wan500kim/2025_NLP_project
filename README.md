
# Game Term Helper AI


### NLP 기반 게임 용어 분석 및 정의 시스템
게임 관련 텍스트에서 전문 용어를 식별하고, 정확한 정의를 제공하는 AI 시스템입니다.

충북대학교 컴퓨터공학과 2025 NLP 프로젝트

---

### 작품 개요

본 프로젝트는 자연어 처리(NLP) 기술을 활용하여 텍스트 내에 포함된 게임 전문 용어(은어, 약어 포함)를 자동으로 인식하고 그 의미를 제공하는 AI 시스템입니다.
일반적인 사전에는 등재되지 않은 게임 커뮤니티의 용어들을 BERT 기반 모델(KoELECTRA)을 통해 학습하여 높은 정확도로 식별합니다.

- 게임 분야 특화 개체명 인식 (NER)
- 문맥을 고려한 용어 식별
- 용어 사전 매핑을 통한 상세 정의 제공
- 텍스트 분석 및 정보 추출 자동화

### 시스템 기능

- **용어 분석 엔진:** 
입력된 문장에서 게임 용어를 실시간으로 추출합니다. 단순 키워드 매칭이 아닌, 딥러닝 모델을 통해 문맥상 게임 용어로 쓰인 경우를 구별합니다.

- **사전 정의 매핑:** 
식별된 용어에 대해 미리 구축된 용어 사전(Dictionary)과 매핑하여, 사용자가 이해하기 쉬운 정의를 출력합니다.
분야(Facet), 게임명 등의 메타데이터를 함께 제공하여 정보의 정확성을 높입니다.

- **배치 처리 및 CLI:** 
대량의 문장을 한 번에 분석하거나, 간단한 명령 줄 인터페이스(CLI)를 통해 즉각적인 테스트가 가능합니다.
